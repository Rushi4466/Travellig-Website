# 🌍 RT SkyRoutes

A modern travel website built using **Node.js, Express, and EJS** showcasing beautiful destinations with a clean and responsive UI.

---

## 🚀 Features

- 🌄 Attractive background with travel theme
- 🧭 Navbar with logo and tagline
- ✈️ "Explore Now" smooth scroll button
- 🏝️ Destination cards with images
- 📞 Contact section with phone & email
- 🎨 Modern UI with hover effects
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- HTML5
- CSS3
